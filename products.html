1D Barcodes

Full support for EAN-13, EAN-8, UPC-A, Code 39, Code 128, Code 128 and ITF-14. Real-time check digit validation with visual feedback as you type.

2D & QR Codes

Generate QR Codes, DataMatrix, QR Code and PDF417 for modern retail and supply chain applications. Toggle between 1D and 2D with one tap.

Camera Scanner

Built-in camera scanner with zoom control and torch support. Scan any barcode and regenerate it instantly in any format. History is auto-saved.

Export Formats

Export every barcode as SVG, PDF, PNG or EPS. Choose 300, 600 or 1200 DPI for PNG. Batch export multiple barcodes as a ZIP file in one tap.

Batch CSV Import

Import a CSV file to generate hundreds of barcodes in one pass. Ideal for product launches, relabeling workflows, and inventory resets. (Pro feature)

History & Favorites

Every generated barcode is automatically saved. Favorite barcodes for quick access. Add notes. Batch export or print label directly from Favorites.
